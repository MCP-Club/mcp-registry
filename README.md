## Servers

### Airtable Server

MCP server for managing Airtable bases and tables with comprehensive API support

**Details:**
```json
{
    "command": "npx",
    "args": ["-y", "@modelcontextprotocol/server-airtable"],
    "type": "stdio"
}
```
